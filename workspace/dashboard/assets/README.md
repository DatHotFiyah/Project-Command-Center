# Cosmo Ops Floor Assets

Put generated PNG sprite sheets and cutout assets here for the diorama/project-management interface.

Suggested structure:
- `characters/` for team/dev unit sprites
- `furniture/` for desks, monitors, chairs, props
- `tiles/` for floor/walls/room pieces
- `fx/` for glow effects, particles, ambient overlays
- `ui/` for icons, badges, bars, markers

Preferred format:
- PNG with transparency
- top-down or slightly angled top-down perspective
- clean readable silhouettes
- consistent scale across sheets
